# nextjs-training

NextJS training plan

## Overview
Generally, this revision of Next.js training plan which focuses on the latest version of Next.js ([15.0.3](https://github.com/vercel/next.js/releases/tag/v15.0.3)). The trainees can discover the further details in the course https://nextjs.org/learn/. As a result, the trainees can follow this instruction to learn Next.js fundamentals step by step and hands-on practices.
Furthermore, the trainees can invest more time on the official document https://nextjs.org/docs to have understanding in depth about Next.js key features.

## Reference Timeline
Total: 3 weeks
- Fundamentals Approach
- Advanced Approach
- Practice Improvements

## Prerequisite knowledge
This course assumes you have a basic understanding of HTML/CSS, JavaScript and React. If you're new to React, we recommend going through our [React Foundations](https://nextjs.org/learn/react-foundations) (React [18.3.1](https://github.com/facebook/react/releases/tag/v18.3.1) or latest version) course first to learn the fundamentals of React, such as components, props, state, and hooks, and newer features like Server Components and Suspense.

## System requirements
Before you start this course, make sure your system meets the following requirements:
- Node.js 20 or later installed.
- Operating systems: macOS, Windows (including WSL), or Linux.
- [Vercel](https://vercel.com/) must be included in this section as a cloud platform which will be in charge of hosting the output of practice.
